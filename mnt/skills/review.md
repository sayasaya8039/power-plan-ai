# /review スキル

コードの品質チェックと改善提案を行います。詳細なルールは `../../.claude/rules/code-review.md` を参照。

## 実行内容

1. **対象ファイルの特定**
   - 引数で指定されたファイル、またはステージング済みファイル
   - 指定がない場合は直近で変更されたファイル

2. **コードレビュー実行**
   - `code-review.md` の観点でレビュー
   - 正確性、可読性、保守性、パフォーマンス、セキュリティを確認

3. **レビュー結果を出力**
   - 総合評価（✅ 承認 / ⚠️ 要修正 / ❌ 却下）
   - 良い点と指摘事項をリスト化

## オプション

```bash
/review                    # ステージング済みファイルをレビュー
/review src/index.ts       # 特定ファイルをレビュー
/review --security         # セキュリティ重点レビュー
/review --performance      # パフォーマンス重点レビュー
```
